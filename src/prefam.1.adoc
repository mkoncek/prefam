== NAME
prefam - preloaded file access monitor

== SYNOPSIS
*PREFAM_OUTPUT_FD*=_fd_ *LD_PRELOAD*=_libdir_/libprefam.so _program_ [args...]

== DESCRIPTION
_Prefam_ is a library that overrides system calls `open*`, `exec*`, `posix_spawn*` and records which file paths have been accessed by these system calls.

The library writes accessed file paths into the provided file descriptor as soon as the events happen.
The format of the output is a line-separated list of file paths.

== ENVIRONMENT VARIABLES
PREFAM_OUTPUT_FD:: The numeric file descriptor into which the library writes the accessed paths.
