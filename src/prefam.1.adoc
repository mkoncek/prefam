== NAME
prefam - preloaded file access monitor

== SYNOPSIS
*PREFAM_OUTPUT_FD*=_fd_ *LD_PRELOAD*=_libdir_/libprefam.so _program_ [args...]

== DESCRIPTION
_Prefam_ is a library that overrides system calls `open*`, `exec*`, `posix_spawn*`, records which file paths have been accessed by these system calls and outputs the paths.

== ENVIRONMENT VARIABLES
PREFAM_OUTPUT_FD:: The numeric file descriptor into which the library writes the accessed paths.
