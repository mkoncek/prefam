= Prefam
:source-highlighter: rouge

Preloaded file access monitor.

_Prefam_ is a library that overrides system calls `open*`, `exec*`, `posix_spawn*` and records which file paths have been accessed by these system calls.

== Usage

In order to use this library you need to preload it and pass it the output file descriptor number via an environment variable `PREFAM_OUTPUT_FD`.

The library writes accessed file paths into the provided file descriptor as soon as the events happen.
The format of the output is a line-separated list of file paths.

.Example usage from Bash:
[source,bash]
----
export PREFAM_OUTPUT_FD=3
exec {PREFAM_OUTPUT_FD}>output.log

LD_PRELOAD=${libdir}/libprefam.so.1 command [args...]

exec {PREFAM_OUTPUT_FD}>&-
----

== Building

Build the project:

----
$ make compile
----

Build manual page:

----
$ make manpage
----

Run tests:

----
$ make test
----
