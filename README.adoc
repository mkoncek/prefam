= Prefam

Preloaded file access monitor.

== Usage

In order to use this library you need to preload it and pass it the output file descriptor number via an environment variable `PREFAM_OUTPUT_FD`.

The library writes accessed file paths into the provided file descriptor as soon as the events happen.
The format of the output is a line-separated list of file paths.

.Example usage from command line:
----
PREFAM_OUTPUT_FD=${fd} LD_PRELOAD=${libdir}/libprefam.so program [args...]
----

== Building

Build the project:

----
$ make compile
----

Build manual page:

----
$ make manpage
----

Run tests:

----
$ make test
----
